# Manuscript Submission Review Notes

This review treats the current manuscript as a scientific submission draft and identifies the main changes still needed before the paper should be considered submission-ready.

1. Add a compact data-provenance table that maps each pressure and speciation source to temperature, loading, MEA mass fraction, observable type, and row count.
2. Add a residual summary table for pressure and speciation metrics, including accepted rows, excluded upper-bound rows, and direct-positive rows.
3. Add an activity-coupled pressure figure analogous to the Phase 1 pressure plot so the 0.493 pressure residual is visually auditable.
4. Add a short reaction-basis table listing the five solved reactions, equilibrium constants, units, and temperature convention.
5. Add an explicit source-basis note for each reaction constant so readers can trace the ideal and activity-coupled calculations to literature values.
6. Add uncertainty or sensitivity bands for the activity-coupled speciation curves once parameter uncertainty is available.
7. Add a parameter-correlation or local-sensitivity table for the fitted MEAH+ and MEACOO- ion parameters.
8. Add a source-stratified residual comparison so readers can see whether specific literature families dominate the pressure or speciation errors.
9. Move the long literature-comparison table to supporting information if the target journal has strict article-length expectations.
10. Add a schematic of the true-species calculation showing apparent inputs, charge balance, reaction equilibrium, activities, and pressure calculation.
11. Add a short validation paragraph for water-balance and charge-balance closure in the assembled speciation rows.
12. Add a clear distinction between direct-positive bicarbonate targets, reported-zero bicarbonate rows, and balance-inferred rows in a table rather than only prose.
13. Add an archived software/data release DOI before submission so the data-availability statement points to a stable record.
14. Confirm author affiliations, funding, conflicts, acknowledgments, and journal-specific declarations before exporting the final submission package.
15. Recheck all figures at final journal column width after template selection, especially the dense parameter tables and two-panel amine-ion regression figure.
