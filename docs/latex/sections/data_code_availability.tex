\section*{Data and Code Availability}
The processed validation tables, fitted parameter tables, plotting data, and source identifiers used in this manuscript are distributed with the source package.  The manuscript figures were generated from recorded CSV inputs, and the cited literature sources identify the original VLE, speciation, and electrolyte-model data.  No archival DOI has been minted for the source package at the time of manuscript preparation; the cited release should be used for reproduction until an archive is issued.
