\section{MEA System Workflow and Results}\label{sec:mea-system-modeling-results}
This section reports the evidence generated for the \MEA--\COtwo--\HtwoO system.  The results are organized from data assembly to baseline checks, neutral ePC-SAFT parity, full ionic ePC-SAFT pressure/speciation evaluation, and parameter-regression readiness.

\subsection{Experimental Data Basis}
The VLE data set contains 161 \COtwo partial-pressure records from Aronu et al., Hilliard, Jou et al., Mamun et al., and Xu et al.\cite{Aronu2011,Hilliard2008,Jou1995,Mamun2005,Xu2011}.  Each record contains amine mass fraction, temperature, \COtwo loading, and measured \COtwo pressure.  The speciation data set contains 74 NMR-derived liquid-composition records from Boettinger et al., Jakobsen et al., and Matin et al.\cite{Bottinger2008,Jakobsen2005,Matin2012}.  The data are expressed on a common loading basis and reconciled to satisfy carbon, amine, water, and charge balances before ePC-SAFT evaluation.

\subsection{Baseline and Neutral ePC-SAFT Checks}
Two baseline checks were used before interpreting the full ionic model.  First, the legacy six-species chemistry path was regenerated for 30 wt\% \MEA.  It includes \COtwo, \MEA, \HtwoO, \MEAH, \MEACOO, \HCOthree, and the aggregate $\MEA+\MEAH$.  The pressure comparison against the Jou 30 wt\% data gives median absolute logarithmic residuals of approximately 0.39, 0.31, 0.15, 0.06, and 0.08 at 40, 60, 80, 100, and \SI{120}{\degreeCelsius}, respectively.  This reproduces the expected pressure trend and provides continuity with the older PC-SAFT path.

Second, the neutral ePC-SAFT pressure calculation was compared directly with the legacy PC-SAFT pressure output.  The neutral parity check demonstrates that the ePC-SAFT package can reproduce the neutral pressure path when the electrolyte terms are not the controlling difference.  This isolates the later ionic comparisons from simple package-interface or unit-conversion errors.

\subsection{Full-Ionic ePC-SAFT Parameter Set}
The full ionic liquid basis contains nine species: \COtwo, \MEA, \HtwoO, \MEAH, \MEACOO, \HCOthree, \COthree, \HthreeO, and \Hydroxide.  The working ePC-SAFT parameter set uses existing neutral and ion entries for \COtwo, \HtwoO, bicarbonate, carbonate, hydronium, and hydroxide, while exposing \MEA, \MEAH, \MEACOO, and selected binary terms as regression degrees of freedom.

The binary-interaction matrix currently retains only one nonzero seed value, $k_{\MEA,\HtwoO}=-0.052$, while the most important future regression candidates include \COtwo--\MEA, \COtwo--\MEAH, \COtwo--\MEACOO, \MEAH--\MEACOO, and \MEAH--\HCOthree.  This sparse seed is useful for diagnosis because it exposes the effect of the ionic/activity model before a heavily tuned parameter set is introduced.

\subsection{Full-Ionic Pressure and Speciation Evaluation}
The full ionic ePC-SAFT evaluation solves all 161 assembled VLE records and all 74 speciation records without recorded state failures.  The pressure residuals have a median absolute value of 0.362 on the $\log_{10}(P_{\mathrm{model}}/P_{\mathrm{data}})$ scale, with a maximum absolute value of 0.973.  These values show that the seed parameter set has the correct order of magnitude across the literature data, but it still requires formal regression before it can be presented as a final predictive model.

The speciation comparison is shown in \cref{fig:ionic-speciation}.  Major amine-family species are reproduced more closely than trace molecular \COtwo, hydronium, and hydroxide.  The median absolute logarithmic model/data deviations are approximately 0.163 for \MEA, 0.0087 for \HtwoO, 0.0876 for \MEAH, 0.0460 for \MEACOO, 0.454 for \HCOthree, and 0.777 for \COthree.  The much larger trace-species deviations for molecular \COtwo, \HthreeO, and \Hydroxide reflect the difficulty of comparing species that are either indirectly inferred or present near the numerical floor.  They are therefore diagnostic residuals rather than the primary regression-quality claim.

\subsection{Regression Status and Interpretation}
The parameter-regression workflow has been formulated as a bounded least-squares problem over pressure, speciation, activity-based reaction residuals, and regularization to the seed parameter vector.  The currently available parameter artifact should be interpreted as a regression-ready seed set.  It demonstrates that the full-ionic ePC-SAFT calls, reactive speciation equations, advanced Born options, pressure prediction, and artifact generation can be executed consistently.  It does not yet establish a converged optimum for \MEA, \MEAH, \MEACOO, or the binary interactions.

This distinction is important scientifically.  The present figures show the feasibility and current performance of the full ionic model structure, not the endpoint of parameter estimation.  The final parameter study should report the optimized parameter vector, active bounds, objective-function decrease, pressure residual distribution, speciation residual distribution, and sensitivity of the fitted result to the selected binary-interaction set.
