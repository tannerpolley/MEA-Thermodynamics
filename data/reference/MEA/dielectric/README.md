# MEA-H2O Dielectric Extraction Targets

This folder is reserved for machine-readable relative-permittivity data or correlations for MEA-H2O mixtures.

Current status: literature lead only. Repo-local digests mention dielectric handling through Uyan, Cleeton, Schick, Hsieh, Floriano, and Nascimento, but no MEA-H2O extraction table has been promoted here yet.

Do not promote MEA `f_solv` from pressure data alone.
