\begin{abstract}
    Post-combustion carbon capture (PCC) using aqueous monoethanolamine (\MEA) remains one of the most mature routes for removing \COtwo from dilute flue gas streams. Reliable process simulation of this solvent requires a thermodynamic model that can represent vapor-liquid equilibrium, reactive speciation, and electrolyte effects over the loading range encountered in absorbers and strippers. This work develops the structure for an ePC-SAFT description of the aqueous \MEA--\COtwo--\HtwoO system. Experimental VLE and NMR speciation data are organized into a common loading convention, converted into six-species and nine-species reactive liquid bases, and used to evaluate baseline pressure and speciation diagnostics. The nine-species basis explicitly includes \MEA, \MEAH, \MEACOO, \HCOthree, carbonate, hydronium, and hydroxide, providing the composition framework required for electrolyte PC-SAFT parameter regression. The current results establish a stable data and modeling workflow, identify the parameters and residual definitions needed for the final ePC-SAFT fit, and define the thermodynamic foundation for later use in an absorption-column simulation.
\end{abstract}
