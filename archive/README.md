# Archive

This folder contains historical scripts, data copies, and old generated artifacts that are no longer maintained entrypoints for the project.

Archived files are reference-only. The active workflow is documented in the root `README.md` and runs through package-native `uv run python -m MEA...` commands.

Historical MEA command names such as `Get_True_Mol_Frac*.py`, `PC_SAFT_test.py`, and `legacy_pcsaft_baseline.py` are archived here so older work can still be inspected. They are not maintained entrypoints.
